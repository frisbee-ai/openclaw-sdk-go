## Sampling Rate

- **After goreleaser config change:** Run `goreleaser check` to validate YAML syntax + schema
- **After tag simulation:** Verify tag format with `git tag -l 'v*.*.*'`
- **After changelog change:** Verify `cliff.toml` with `git-cliff --config cliff.toml --verify`
- **Before `/gsd:verify-work`:** Full `goreleaser check` must pass
