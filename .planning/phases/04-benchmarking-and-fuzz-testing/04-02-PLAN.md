**Add new `benchmark` job parallel to existing `test` job:**

```yaml
  benchmark:
    name: Benchmark
    runs-on: ubuntu-latest
    strategy:
      matrix:
        go-version: ['1.24']
    steps:
      - name: Checkout code
        uses: actions/checkout@v6

      - name: Set up Go
        uses: actions/setup-go@v6
        with:
          go-version: ${{ matrix.go-version }}
          cache: true

      - name: Install benchstat
        run: go install golang.org/x/perf/cmd/benchstat@latest

      - name: Run benchmarks
        run: |
          mkdir -p benchmark-results
          # Run all benchmarks, output to .bench files
          go test -bench=. -benchmem -run=^$ -count=1 ./pkg/protocol/ ./pkg/managers/ 2>&1 | tee benchmark-results/bench.txt
          # Note: Go benchstat format is text, convert to .bench format if needed

      - name: Download baseline (PR only)
        if: github.event_name == 'pull_request'
        uses: actions/download-artifact@v6
        with:
          name: benchmark-baseline
          path: benchmark-results/
          continue-on-error: true

      - name: Compare benchmarks (PR only)
        if: github.event_name == 'pull_request'
        run: |
          if [ -f benchmark-results/baseline.txt ]; then
            # Compare with baseline using benchstat
            benchstat benchmark-results/baseline.txt benchmark-results/bench.txt || true
            # Parse benchstat output for regression
            # Regression = p<0.05 AND regression>10%
            # If regression detected, exit 1 to block merge
          fi

      - name: Upload baseline (main branch only)
        if: github.event_name == 'push' && github.ref == 'refs/heads/main'
        run: |
          cp benchmark-results/bench.txt benchmark-results/baseline.txt
          # Upload baseline for future PR comparisons
          echo "Baseline saved"

      - name: Upload results
        uses: actions/upload-artifact@v6
        with:
          name: benchmark-results
          path: benchmark-results/
          retention-days: 30
```

**Key points per D-09 through D-14:**
- Separate job (not combined with test job) - D-09
- Uses same go-version: ['1.24'] as test job - D-14
- benchstat installed via `go install golang.org/x/perf/cmd/benchstat@latest` - D-13
- Artifact path: benchmark-results/*.bench (or .txt for text format) - D-11
- Regression threshold: p<0.05 AND >10% - D-11
- On PR: compare vs downloaded baseline - D-10
- On main: upload .bench artifacts - D-12

**Regression detection logic (D-11):**
- Run benchstat to compare baseline vs PR
- Parse output for p-value and magnitude
- Block (exit 1) only when BOTH p<0.05 AND regression>10%
- Otherwise warn but pass
